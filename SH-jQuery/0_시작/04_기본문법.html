jQuery 기본 문법

$('selector').attribute;

$('selector').method();

$
- jQuery 정의 및 접근(access)
('selector')
- 선택자
객체 또는 HTML 요소(문자열로 처리)
.attribute / .method()
- 해당 객체(요소)에 속성 또는 행동 지정(설정)

jquery 작성 예

  • $(this).hide(); //현재 요소를 숨겨라!
  • $('p').hide(); //(모든) 'p' 태그를 숨겨라!
  • $('.cls').hide(); //(모든) 'cls' 클래스를 숨겨라!
  • $('#i').hide(); //아이디 'i'를 숨겨라!
