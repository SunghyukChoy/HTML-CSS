브라켓에서 주석 넣기

주석은 ctrl+/ 로 작성합니다.

또 다른 주석은 crtl + shift + / 로 작성

브라켓의 사이드바를 보여주거나 숨기려면 ctrl + alt + h

콘텐츠 복제는 ctrl + d

콘텐츠 이동은 ctrl + shift + 상하화살표키

동일한 태그를 여러 개 작성하는 방식 (*)

p*3을 쓰면 p태그가 3개 만들어진다.

자식 태그 만들기

div>p를 쓰면 div의 자식인 p가 만들어진다.

아이디 속성 만들기

태그명#아이디값

클래스 속성 만들기

태그명.클래스명

일반 속성 만들기

태그명[속성명=속성값]

번호 붙이기

$는 일련번호를 붙여준다.

시작값은 $서식 뒤에 @시작값으로 지정한다.(ex. $@2)

콘텐츠 삽입

콘텐츠1

콘텐츠2

콘텐츠3
